# Typescript tutorial

## install
    sudo npm i -g typescript

## compile typescript to javascript
    tsc sandbox.ts -w

## project folder structure
    tsc --init
    vi tsconfig.json
    rootDir (TS code) - ./src
    outDir (JS code) - ./public
